# 🍱 Tiffin Service Dashboard

A modern and scalable **Tiffin Management System Dashboard** built using React and Tailwind CSS.  
This project helps manage subscriptions, users, meals, payments, and analytics efficiently.

---

## 🚀 Features

### 📊 Dashboard
- Orders overview
- Revenue tracking
- Customer insights
- Interactive charts

### 👥 User Management
- Add/Edit/Delete users
- Role-based system (Admin / Customer)
- Assign subscription plans

### 📦 Subscription System
- Multiple plans (Basic, Standard, Premium)
- Active & expired subscriptions tracking

### 🍱 Meal Planner
- Weekly meal schedule
- Breakfast / Lunch / Dinner planning

### 🚚 Delivery Tracking
- Order status (Delivered / Pending / Cancelled)
- Delivery logs

### 💳 Payments
- Revenue summary
- Paid / Pending tracking

### 📈 Analytics
- Orders over time
- Daily distribution
- Business insights

### 💬 Feedback System
- Meal ratings
- Customer feedback
